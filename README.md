# st
Il mio fork di <a href="https://st.suckless.org">st (*simple terminal*)</a> con alcune modifiche e patchs
## patch
- **alpha**
- **color emoji support**
- **scroll integration**
- **xrdb color support**
## Notes on Emojis and Special Characters
If st crashes when viewing emojis, install <a href="https://aur.archlinux.org/packages/libxft-bgra/">libxft-bgra</a> from the AUR.
